# Portal RRHH FZ

Webapp interna V1 para gestion de colaboradores, expedientes digitales, alertas de vencimiento, dashboard, usuarios y roles.

Stack:
- Backend: ASP.NET Core Web API
- Frontend: React + Vite + TypeScript
- Base de datos: SQL Server
- ORM: Entity Framework Core
- Autenticacion: JWT Bearer

La importacion real de colaboradores queda fuera de esta etapa. La herramienta de migracion simula por defecto y solo escribe con `--confirm`.
